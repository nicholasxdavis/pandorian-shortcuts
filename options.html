Keyboard Shortcuts

Ctrl+K (or Cmd+K on Mac) opens the Command Palette to quickly search and activate any shortcut.

Ctrl+Q (or Ctrl+Shift+Q on Mac) generates a QR code for the current page instantly, even when the extension popup is closed.

Ctrl+V (or Cmd+V on Mac) opens/closes the Bookmark Sidebar.

Ctrl+B (or Cmd+B on Mac) bookmarks the current page instantly.

You can also assign direct keyboard shortcuts (Ctrl+Shift+1-2) to your favorite shortcuts using the "Assign" button next to each shortcut. All keyboard shortcuts can be customized in Chrome's extension keyboard shortcuts settings.
